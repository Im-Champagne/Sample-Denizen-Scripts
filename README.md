## ðŸ› ï¸ How to Use These Scripts

Using these scripts is super simple! If you're new to Denizen, you should be checking out [`[Denizen Beginner's Guide]`](https://guide.denizenscript.com) to familiarize yourself with setting up scripts.

1.  **Prerequisites:** You must have **Denizen** installed on your Spigot/Paper server.
2.  **Installation:** Save the DenizenScript code to a `.dsc` file located in your server's `plugins/Denizen/scripts/` folder.
3.  **Reload:** Run the command `/ex reload` in-game or in the console.
